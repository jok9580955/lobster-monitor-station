隐私政策 · 最后更新：2026年3月25日

概述

「龙虾监测站」（以下简称"本应用"）非常重视您的隐私。本隐私政策旨在向您说明我们如何处理您的数据。

✅ 本应用 不收集、不上传任何个人信息。所有数据均存储在您的设备本地，不会发送至任何第三方服务器。

我们收集的信息

本应用仅在您的设备本地存储以下信息，不会上传至任何服务器：

  • 连接配置：您填写的 Agent 主机地址和 API Key，通过系统 UserDefaults 本地保存。
  • 技能偏好：您安装或卸载的技能插件选项，本地保存。
  • 告警阈值设置：CPU 及 Token 使用率的告警阈值，本地保存。

我们不收集的信息

  • 不收集您的姓名、手机号、邮箱或任何身份信息
  • 不收集位置信息
  • 不收集设备标识符（IDFA / IDFV）
  • 不使用任何第三方分析或广告 SDK
  • 不在应用内展示广告

网络请求

本应用仅在以下情况下发起网络请求：

  • 当您配置了本地运行的 OpenClaw Agent 服务地址，应用将与该地址通信以获取状态数据。整个通信过程在您的私有局域网内完成，不涉及公共互联网上的第三方服务器。

数据安全

您的 API Key 通过 iOS 系统的 UserDefaults 存储。我们建议您不要在不受信任的设备上存储高权限 API Key。

儿童隐私

本应用不面向13岁以下儿童，也不会有意收集儿童个人信息。

政策变更

如本隐私政策发生重大变更，我们将在应用内或通过 App Store 更新说明告知用户。更新后继续使用本应用，即视为您接受新版隐私政策。
