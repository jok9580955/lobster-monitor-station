隐私政策 · 最后更新：2026年4月3日

概述

「龙虾 AI 导航」（以下简称"本应用"）非常重视您的隐私。本隐私政策旨在向您说明我们如何处理您的数据。

✅ 本应用 不收集、不上传任何个人信息。所有数据均存储在您的设备本地，或通过您的授权仅在私有服务间通信。

我们收集的信息

本应用仅在您的设备本地存储以下信息，不会上传至任何我们的服务器：

  • 接口配置：您填写的 AI 代理地址和私有 API Key，通过系统 UserDefaults 本地保存。
  • 收藏偏好：您收藏的 AI 工具列表，本地保存。
  • 应用设置：外观模式、推送开关等偏好设置，本地保存。

我们不收集的信息

  • 不收集您的姓名、手机号、邮箱或任何身份信息
  • 不收集位置信息
  • 不收集设备标识符（IDFA / IDFV）
  • 不使用任何第三方侵入式分析 SDK
  • 不在应用内展示第三方广告

网络请求

本应用仅在以下情况下发起网络请求：

  • 当您配置了私有 AI 代理服务地址，应用将与该地址通信以获取相关 AI 服务数据。
  • 用于获取最新的 AI 工具推荐列表（仅限匿名元数据）。

数据安全

您的 API Key 通过 iOS 系统本地加密存储。我们强烈建议您不要在公共或不受信任的设备上使用本应用。

政策变更

如本隐私政策发生变更，我们将在应用内或通过 App Store 更新说明告知。继续使用本应用即视为您接受新版隐私政策。
